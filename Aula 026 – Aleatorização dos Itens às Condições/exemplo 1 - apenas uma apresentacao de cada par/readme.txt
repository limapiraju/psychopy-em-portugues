Como designar aleatoriamente os itens às diferentes condições experimentais? Em outras palavras, como fazer uma designação dos itens às condições de uma forma que essa designação seja, potencialmente, personalizada para cada participante? Este programa apresenta uma maneira de atingir esse objetivo. A descrição completa do programa é apresentada na Aula 026 do YouTube (https://www.youtube.com/watch?v=lG1faMnd6LI).
