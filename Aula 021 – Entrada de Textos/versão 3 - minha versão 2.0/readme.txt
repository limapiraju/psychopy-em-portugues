Este programa não foi originalmente apresentado na vídeo-aula do YouTube. Nele, eu fiz uma melhoria: substituí a lista buttons por um  dicionário de mesmo nome. Basicamente, um dicionário funciona associando diferentes "chaves" a seus respectivos valores. Um exemplo disso é a chave "comma" e o valor ",". Com o uso do dicionário, pude controlar, com apenas um novo IF, qual valor deve ser substituído no texto, caso o usuário pressione determinada tecla. Veja o código para mais detalhes.
